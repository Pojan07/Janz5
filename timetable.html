ITT 569

Internet of Things

This course provides an introduction to the Internet of Things (IoT) and covers the basic concepts, architecture, protocols, and technologies used in the design and development of IoT systems. Students will learn about the various components of IoT systems, including sensors, actuators, communication modules, and cloud platforms. The course will also cover the challenges and opportunities associated with IoT, such as security, privacy, scalability, and interoperability.

Lecturer: Shapina Binti Abdullah

ELC 550

English For Academic Writing

English for Academic Writing is a course designed to help students develop their writing skills for academic purposes. The course covers a range of topics, including academic style and tone, structure and organization of academic writing, critical thinking and analysis, and citing and referencing sources correctly. Students will learn to write effective essays, reports, research papers, and other academic texts, using appropriate language and vocabulary for their field of study.

Lecturer: Siti Fauziana Binti Zakaria

ITS 462

Fundamental of Artificial Intelligence

Fundamentals of Artificial Intelligence is a course that provides an introduction to the field of AI. The course covers the basic concepts, techniques, and algorithms used in AI. Students will learn about problem-solving and search algorithms, knowledge representation and reasoning, machine learning, natural language processing, and robotics. They will also explore various applications of AI, such as computer vision, speech recognition, and game playing.

Lecturer: Dr. Farah Aqilah Bt Bohani

CSC 574

Dynamic Web Application Development

Dynamic Web Application Development is a course that covers the principles and technologies used to build dynamic web applications. Students learn how to design and develop web applications that interact with databases, manage user authentication and authorization, and provide responsive user interfaces using modern front-end frameworks.

Lecturer: Nur Aina Khadijah Binti Adnan

TMC 451

MANDARIN (LEVEL II)

This course builds on the skills learned in Mandarin (Level I) and expands the student's knowledge of spoken and written Mandarin Chinese. Students will continue to develop their listening, speaking, reading, and writing skills, as well as their understanding of Chinese grammar, vocabulary, and culture.

Lecturer: Leona Kiu King Chieh

ITT 557

Web Application Security

The course includes topics such as threat modeling, input validation, authentication and authorization, session management, access control, cryptography, and secure coding practices. Students will learn about common web application vulnerabilities, such as SQL injection, cross-site scripting (XSS), and cross-site request forgery (CSRF), and how to prevent and mitigate these vulnerabilities.

Lecturer: Dr. Mohsen Bin Mohamad Hatta

ITT588

Front End Technology

Students will learn how to create interactive and visually appealing web pages and web applications, as well as best practices for optimizing performance and accessibility. The course may also cover topics such as responsive design, cross-browser compatibility, and web accessibility standards.

Lecturer: Kamarul Ariffin Bin Abdul Basit
